Files placed in the /plugins folder are all Javascript and jQuery plugins used in this template.
The 'custom.js' file contains custom script.
The 'plugins-setup.js' file contains settings for plugins.
The 'preloader.js' file contains scripts needed for preloader animations and initialization some plugins.
The 'style-switcher.js' file contains all scripts needed for customization the template (changing color theme, background of page and so on).
